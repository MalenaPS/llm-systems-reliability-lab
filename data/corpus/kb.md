# KB (Synthetic)

## BM25
BM25 is a ranking function used in information retrieval to estimate the relevance of documents to a given search query.

## Prompt Injection
Prompt injection is an attack where malicious instructions are inserted to override the system or developer instructions.

## Tool Allowlist
A tool allowlist restricts which tools the system can call, reducing tool hijacking risk.

## Insufficient Evidence
If the retriever returns no relevant evidence, the system must set insufficient_evidence=true and avoid hallucinating.

## Circuit Breaker
A circuit breaker stops calling a failing dependency temporarily to prevent cascading failures.
