# Failure Modes

This document describes the failure modes the system is designed to detect.

---

## Tool failures

Examples:

• transient tool failure  
• hard tool failure  
• malformed tool output

---

## LLM failures

Examples:

• schema-breaking outputs  
• hallucinated evidence  
• policy violations

---

## Adversarial attacks

Examples:

• prompt injection  
• context leakage  
• tool hijacking

---

## Drift failures

Examples:

• model update changes tool usage  
• schema compliance degradation  
• policy adherence degradation
